# 🌱 RenovaRed

> Transformando residuos en oportunidades ♻️

RenovaRed es una plataforma web diseñada para impulsar la **economía circular**, permitiendo a las personas publicar, reutilizar e intercambiar materiales dentro de su comunidad.

---

## ✨ ¿Por qué RenovaRed?

En el modelo tradicional, los residuos terminan descartados.  
RenovaRed propone una alternativa:

➡️ **Convertir lo que no usas en un recurso para otros**

La plataforma conecta usuarios que desean:
- Reducir desperdicios
- Reutilizar materiales
- Generar valor a partir de recursos existentes

## 🧩 Funcionalidades

### 📦 Publicación de materiales
- Creación de publicaciones con:
  - Título
  - Descripción
  - Categoría
  - Ubicación
  - Imagen
  - Precio (opcional)
- Edición y eliminación de publicaciones

### 🔍 Exploración tipo marketplace
- Visualización de materiales disponibles
- Interfaz clara y moderna
- Enfoque centrado en la experiencia del usuario

### 🖼️ Gestión de imágenes
- Subida de imágenes
- Preview en tiempo real
- Manejo dinámico desde el frontend

### 🏷️ Sistema de categorías
- Organización de materiales
- Navegación más eficiente

### 👤 Gestión de usuarios
- Autenticación
- Interacción con publicaciones
