# RenovaRed - Documentación del BACKEND

## Tecnologías
- Node.js v22.17.1
- Express 5.2+
- CORS habilitado

## Instalación
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

## Documentación

La API cuenta con documentación automática usando Swagger:

- **Swagger UI**: `http://localhost:3000/api-docs`
- **Especificación JSON**: `http://localhost:3000/api-docs.json`
